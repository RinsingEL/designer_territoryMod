# 主模块建造范围

## 目标

当前主模块建造主链已经切到：

- `C7` 工头计划
- `C8` 节点决策与程序校验
- `C9` 队列生成与建造执行

主模块建造方案继续围绕这条主链推进，不再回到旧的“排列方式枚举继续扩厚”路线。

## 当前有效结论

- 图像只在总规阶段使用一次
- 后续逐节点阶段只使用结构化数据
- AI 负责当前节点决策
- 程序负责校验、状态维护、错误回包和队列推进
- `C9` 负责消费已校验节点，并进入运行时建造链路

## 当前不做

- 不再扩展 `arrangement_type` 作为主参数
- 不再继续加厚 `core / wing / terminal / junction` 之类中间语义层
- 不把道路与次级建筑重新并入主模块主链

## 当前依赖

- `C7` 工头计划
- `C8` 节点会话与 MCP 提交
- `C9` BuildQueue 与执行器
