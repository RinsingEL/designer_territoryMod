当前实现仓库中的 `src/main/java/com/user/terra_script/world/city/stage/c4/CitySemanticStages.java` 已经存在 `C4_FunctionPlan.json`，但它承担的是旧 district function / whitelist 语义规划职责。

| 新案子概念 | 建议实现名 |

## LLM 复查实现提醒

LLM 复查建议作为独立 adapter，不直接写入几何计算主流程。

| 复查点 | 建议输入 | 建议输出 |
| --- | --- | --- |
| C1/C2 意图复查 | image2 图、CV color clusters、UrbanIntentMap 摘要、overlay | 颜色映射、功能权重建议、road sketch 语义说明 |
| C4 道路复查 | 道路 overlay、`C4RoadNetwork` 摘要、recognition report | 道路等级语义建议、断路/漏路/过密问题 |
| C5 area 复查 | area overlay、`C5AreaPlan` 摘要、C4 overlay | `layout_profile`、`reserved_strategy`、`split_policy`、`size_band` 建议 |

复查输出只能作为建议或人工评估材料，不能覆盖程序计算出的坐标、拓扑、面积和结构合法性。
