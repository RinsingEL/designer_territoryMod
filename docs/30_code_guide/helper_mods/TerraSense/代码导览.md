## 当前缺口

- 没有断点续扫和失败重试。
- AI 分析和人工审核没有清晰分层。
- 没有脱离 MC 的审核编辑器。
- 没有正式 `review.json`。
- 没有一键导出 StructureBinder 当前 catalog。

## 建议新增模块

| `scan_index` | Java 或 Studio 侧 JSON | 记录 pending/scanned/ai_done/reviewed/failed |

| `catalog_exporter` | Studio/脚本 | 输出 StructureBinder 可消费 catalog |

## 与 StructureBinder 对接

当前 StructureBinder 从 `template_output/C3_5_StructureCatalog.preprocessed.json` 读取结构 catalog。

消费入口包括：

- `CityC35CatalogIO`
- `StructureTemplateQueryService`
- C7 工头计划
- C8 节点会话与 jigsaw 求解
- C9 执行层

TerraSense 第一阶段导出必须优先满足当前 C3.5 catalog，再逐步扩展到 `StructureProfile`。
