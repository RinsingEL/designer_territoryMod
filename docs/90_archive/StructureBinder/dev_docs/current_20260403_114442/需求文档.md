## 用户特别强调的约束

- 使用中文对话。
- 文件默认按 UTF-8 打开和保存。
- 回复中的文件链接使用 markdown 绝对路径，格式固定为 `/c:/...`。
- 方案要比“队列管理 + helper”更细。
- 需要考虑落地前清地表、清 feature、嵌入洞中时清原世界方块等预处理需求。
- 需要考虑“提前预存建造计划，等区块完全加载后再落地”的模式。
