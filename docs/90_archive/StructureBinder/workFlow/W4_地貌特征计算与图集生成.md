| 类 | 方法 | 作用 |
| --- | --- | --- |
| `domain.world.stage.W4Stage` | `execute(StageContext)` | 汇总 region cache 并输出摘要/地形缓存 |
| `domain.world.stage.W4PreviewExporter` | `export(...)` | 输出预览图元数据 |
| `core.workflow.FileStageStatusStore` | `updateProgress(...)` | 更新阶段进度 |

## 实现逻辑

- 检查 `ScanResultHolder.regionCacheMap` 是否存在。
- 遍历 `RegionCache`，提取：
  - region id
  - 边界
  - 宽高
  - step
- 调用 `W4PreviewExporter.export(...)` 生成预览图信息。
- 将摘要写入 `W4_TERRAIN_SUMMARY_JSON`。
- 将 `slopeData`、`roughnessData`、`tpiData` 压缩写入 `W4_TERRAIN_FACTS_DAT`。
- 通过 `FileStageStatusStore` 更新进度。

## 输入

- `StageContext`
- `ScanResultHolder.regionCacheMap`

## 输出

- `W4_TERRAIN_SUMMARY_JSON`
- `W4_TERRAIN_FACTS_DAT`
