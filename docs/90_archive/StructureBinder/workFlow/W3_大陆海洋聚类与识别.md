## 核心类与方法

| 类 | 方法 | 作用 |
| --- | --- | --- |
| `domain.world.stage.W3Stage` | `execute(StageContext)` | 执行大陆/海洋聚类 |
| `domain.world.scan.service.ClusterAnalyzer` | `analyze(...)` | 聚类大陆或海洋区域 |
| `domain.world.scan.service.ClusterAnalyzer` | `expandOceans(...)` | 生成扩展 cluster map |
| `client.data.ScanResultHolder` | `get()` | 读取全局扫描缓存 |

## 实现逻辑

- 从 `ScanResultHolder.lastScanData` 读取全局扫描数据。
- 使用 `ClusterAnalyzer.analyze(...)` 对陆地进行 continent 聚类。
- 使用 `ClusterAnalyzer.analyze(...)` 对海洋进行 ocean 聚类。
- 使用 `ClusterAnalyzer.expandOceans(...)` 构建扩展后的 cluster map。
- 把结果写回：
  - `lastClusters`
  - `lastOceanRegions`
  - `lastClusterMap`
- 将大陆/海洋元数据转成 JSON 并写入制品目录。

## 输入

- `StageContext.server`
- `StageContext.artifacts`
- `ScanResultHolder.lastScanData`

## 输出

- `W3_CONTINENT_META_JSON`
- `W3_OCEAN_META_JSON`

输出字段重点：

- 区域 id
- 面积、平均高度、崎岖度、中心点
- 气候均值和分布
- 群系占比和稀有群系点
