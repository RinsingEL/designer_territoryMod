# C9 Contract

## 当前定位

`C9` 负责把已校验的结构节点转成可执行建造任务，并进入运行时执行链。

## 主链

1. 从 `C8.validated_nodes` 取可执行节点
2. 生成 `BuildTask`
3. 写入 `BuildQueue`
4. 等区块满足执行条件后落地

## 当前关键输出

- `C9_Placement`
- `C9_Decoration`
- `C9_BuildQueue`
- `queue_summary`

## 当前运行时职责

- 父节点依赖检查
- footprint collision
- chunk readiness
- 结构实际落地
- retry / blocked

## 后续扩展入口

- terrain preparation
- build placement service
- runtime validator
- fallback service
