# C8 节点会话配置表

## 入口契约

### `city_c8_generate`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `city_id` | string | 是 | 城市编号 |
| `group_id` | string | 否 | 仅返回并落盘当前 group 会话 |

### `city_c8_submit`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `city_id` | string | 是 | 城市编号 |
| `group_id` | string | 否 | 目标 group |
| `build_area_id` | string | 否 | 目标建造区 |
| `node_id` | string | 是 | 当前提交节点 |
| `selected_template_id` | string | 是 | 当前模板 |
| `selected_connector_dir` | string | 否 | 当前方向 |
| `selected_rotation` | int | 否 | 当前旋转 |
| `x` / `z` | int | 否 | 根节点或特例节点允许直接传入 |
| `terrain_relax_profile` | object | 否 | 地形放宽配置 |

### `city_c8_retry`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `city_id` | string | 是 | 城市编号 |
| `group_id` | string | 否 | 目标 group |
| `build_area_id` | string | 否 | 目标建造区 |
| `node_id` | string | 是 | 需要重试的节点 |

## 当前稳定对象

### `FoundationItem`

| 字段 | 说明 |
| --- | --- |
| `session_state` | 当前会话状态 |
| `queue_summary` | ready/validated/failed/total 摘要 |
| `active_node` | 当前活跃节点 |
| `node_queue` | 待决策节点队列 |
| `validated_nodes` | 已通过校验节点 |
| `failed_attempts` | 历史失败记录 |
| `foundation_context` | 基台、高程与中文说明 |
| `placements` | 兼容保留的旧 placement 列表 |

### `NodeTask`

| 字段 | 说明 |
| --- | --- |
| `node_id` | 稳定节点编号 |
| `parent_node_id` | 父节点编号 |
| `phase_key` / `phase_name` | 所属阶段 |
| `status` | 当前状态 |
| `template_pool_id` | 当前模板池 |
| `candidate_template_ids` | 当前候选模板 |
| `selected_template_id` | 已选模板 |
| `selected_connector_dir` | 已选方向 |
| `selected_rotation` | 已选旋转 |
| `x / y / z` | 当前坐标 |
| `last_error_code / last_error_message` | 最近错误 |
| `terrain_relax_profile` | 当前地形放宽配置 |
| `placement` | 校验成功后生成的 placement |

### `NodeSubmitResult`

| 字段 | 说明 |
| --- | --- |
| `ok` | 是否通过校验 |
| `node` | 当前节点回写结果 |
| `placement` | 新写入的 placement |
| `error_code` | 稳定错误码 |
| `error_message` | 中文错误说明 |

## 文件落盘

| 场景 | 文件 |
| --- | --- |
| 城市级 | `C8_FoundationPlan.json` |
| group 级 | `group/<groupId>/c8_foundation.json` |
| group 调试产物 | `CityC8Stages.saveDebug(...)` 输出 |

## 当前优先级规则

- `city_c8_generate` 优先读取 `group/<groupId>/c7_selection.json`，找不到时才回退城市级 `C7`。
- `city_c8_submit`、`city_c8_retry` 与 `city_jigsaw_solve` 优先读取 `group/<groupId>/c8_foundation.json`，找不到时才回退城市级 `C8`。
