# C7 工头计划配置表

## 入口契约

### `city_c7_generate`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `city_id` | string | 是 | 城市编号 |
| `group_id` | string | 否 | 仅返回并落盘当前 group |
| `strict_tag_source` | boolean | 否 | 是否启用严格标签过滤 |
| `arrangements` | array/object | 否 | 传入后改走 `fromDecisionRequest(...)` |

## 主要回包字段

| 字段 | 说明 |
| --- | --- |
| `selection_count` | 兼容保留的模板选择数量 |
| `arrangement_count` | 兼容保留的排列决策数量 |
| `foreman_plan_count` | 当前工头计划数量 |
| `phase_count` | 当前阶段数量 |
| `catalog_source` | 当前使用的 catalog 来源 |
| `decision_source` | 计划生成来源 |
| `selection_mode` | 当前候选选择模式 |
| `strict_tag_source` | 当前是否启用严格标签过滤 |
| `strict_filter_failure_reason` | 严格过滤失败原因 |
| `file` | 当前落盘文件路径 |

## 当前稳定对象

### `C7Selection`

| 字段 | 说明 |
| --- | --- |
| `version` | 当前契约版本，代码中为 `2` |
| `step` | 固定为 `C7` |
| `foreman_plan` | group 级主工头计划 |
| `foreman_plans` | 城市级工头计划列表 |
| `phase_list` | 当前主工头计划阶段列表 |

### `ForemanPlan`

| 字段 | 说明 |
| --- | --- |
| `group_id` | 所属组编号 |
| `build_area_id` | 所属建造区 |
| `global_goal` | 全局中文目标 |
| `start_node` | 唯一开工节点 |
| `phase_list` | 当前阶段列表 |
| `connector_target_rules` | connector 目标规则 |
| `template_pool_refs` | 模板池引用 |

### `StartNode`

| 字段 | 说明 |
| --- | --- |
| `node_id` | 稳定节点编号 |
| `phase_key` | 所属阶段键 |
| `allowed_connector_dirs` | 允许连接方向 |
| `template_pool_id` | 模板池编号 |
| `candidate_template_ids` | 候选模板列表 |

## 文件落盘

| 场景 | 文件 |
| --- | --- |
| 城市级 | `C7_TemplateSelection.json` |
| group 级 | `group/<groupId>/c7_selection.json` |
| group 校验报告 | `CityC7Validation.save(...)` 产物 |

## 当前兼容说明

- `selections` 与 `arrangements` 继续保留，供旧链路与调试读取。
- 当前主链优先消费 `foreman_plan / foreman_plans / phase_list`。
